Addressbook.app utility
-----------------------
A utility solving a limitation of the CardDAV protocol in some applications. For more information,
see http://kb.kerio.com/article/contacts-folders-in-apple-addressbook-app-via-carddav-954.html

Configuration
-------------
To configure Addressbook utility, open the config.php and enter your user login credentials as you usualy
do in Addressbook.app or in Kerio Connect Client (aka. Webmail).

Security
--------
It is recommended to provide basic secured access to the config.php file,
eventually the entire application, as described in the main documentation.

Other
-----
Sometimes deleting the cache file of the Addressbook.app is recommended.
