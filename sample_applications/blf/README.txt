BLF
---
Busy Lamp field (BLF) feature allows users to monitor several other extensions.
Display your lines from Kerio Operator easily indicating idle, ringing or connected state.

Configuration
-------------
To configure BLF, open the config.php and enter your login credentials with Administrator right.

Security
--------
It is recommended to provide basic secured access to the config.php file,
eventually the entire application, as described in the main documentation.

Other
-----
Make sure that db folder is writeable.
