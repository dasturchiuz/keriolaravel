BACKUP KERIO OPERATOR
---------------------
Simple application with an intuitive form and overview of recent backups.

Configuration
-------------
To configure Backup, open the config.php and enter your login credentials with Administrator right.
The folder with backups is stored in the $backupDir variable.

Security
--------
It is recommended to provide basic secured access to the config.php file  and the backup directory,
eventually the entire application, as described in the main documentation.

Other
-----
Make sure that backup folder is writeable.
