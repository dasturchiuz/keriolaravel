Sample Applications

Here you can find some more complex standalone applications.
Some applications may have their own configuration file, usually config.php.

Please note that these applications are intended for use as a technical demonstration and are not intended for deployment in a production environment.

  •  My Servers
    All your servers at a single spot, simple intuitive dashboard. You can immediately click on a hostname and login to the Web Administration.

  •  Mobile Devices
    Your mobile devices using ActiveSync®. Discover who uses which device, version or just when the device has been last synchronized.

  •  Geolocator
    Get the list of active connections and put them into Google Maps by GeoIP database to find out where the connections are established from.

  •  Busy Lamp Field
    BLF feature allows users to monitor several other extensions. Display your lines from Kerio Operator easily indicating idle, ringing or connected state.

  •   Backup
    Simple application with an intuitive form and overview of recent backups.

  •   Addressbook.app Utility
    A utility solving a limitation of the CardDAV protocol in some applications. For more information, see KB article.

  •   Import to File Library
    Have some files and want to upload them to Kerio Workspace using a script?
